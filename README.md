# Windows 11 Web Clone - Fully Functional HTML/CSS/JS Replica

A pixel-perfect, highly interactive browser-based clone of the Windows 11 desktop environment. Built strictly focusing on mimicking the micro-interactions, responsive side-panels, and UI component behavior of the modern Windows OS using vanilla front-end tech stack.

## ✨ Features Demonstrated in Video
- **Interactive Lock Screen:** Smooth transitions with live time, day, and date indicators.
- **Dynamic File Explorer:** Working navigation for Home, Desktop, Documents, Pictures, and system local drives.
- **Responsive Web Utilities:** Simulated Microsoft Edge landing page, custom fully-functional Notepad, and localized live Weather widgets.
- **System Tray & Flyouts:** Clickable calendar menu pane and full notification/settings bar containing operational toggles (Do Not Disturb, Dark Mode, Volume, and Brightness sliders).
- **Working Start Menu & Search:** Clean layout mapping pinned applications and recommended files.

## 🛠️ Tech Stack
- HTML5 (Semantic Structure)
- CSS3 (Custom styling, modern layout systems, glassmorphism, and transitions)
- JavaScript (DOM manipulation, real-time clock tracking, dynamic state management for windows)

## 📸 Project Architecture
The project structure encapsulates modular windows styling allowing multiple components (Settings, Mail, File Explorer) to run with clean toggle states without overlapping bugs or structural breakage.
